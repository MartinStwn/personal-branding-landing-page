# Personal Branding Landing Page — Martinus Setiawan

Landing page profesional untuk Web Developer & Cyber Security Specialist. Dibangun dengan **Next.js 15**, **React 19**, **Tailwind CSS 4**, dan **shadcn/ui**.

## Fitur

- ⚡ Performa cepat dengan Next.js App Router
- 🎨 Dark mode & glassmorphism design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🔒 Fokus pada layanan Audit Keamanan Website
- 📬 Form kontak terintegrasi WhatsApp
- 🖼️ Portfolio proyek
- ⭐ Testimoni klien

## Teknologi

| Teknologi | Versi |
|-----------|-------|
| Next.js | 15.x |
| React | 19.x |
| Tailwind CSS | 4.x |
| TypeScript | 5.x |
| shadcn/ui | latest |

## Cara Menjalankan

```bash
npm install
npm run dev
# Buka http://localhost:3000
```

## Build & Deploy

```bash
npm run build
npm run start
```

Deployed with [Vercel](https://vercel.com).
